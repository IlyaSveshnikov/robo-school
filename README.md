# ROBO SCHOOL — landing page

> Responsive landing page for a robotics teacher-training school. Built from a Figma design with semantic HTML, modern CSS and vanilla JavaScript — no frameworks, no build step.

**Лендинг школы робототехники** — курсы повышения квалификации по робототехнике и программированию для педагогов начальной школы. Свёрстан по макету из Figma на чистом HTML / CSS / JavaScript.

🔗 **Live demo:** _добавьте ссылку на GitHub Pages после публикации_

## ✨ Возможности

- **Адаптивная вёрстка** — десктоп, планшет и мобильные устройства (брейкпоинты 1085 / 768 / 560 px).
- **Sticky-навигация + scrollspy** — шапка уплотняется и получает blur-фон при скролле, активный пункт меню подсвечивается по текущей секции.
- **Прогресс-бар чтения и кнопка «наверх»** — индикатор прокрутки сверху и плавный возврат к началу.
- **Карусель тренеров** — пошаговая прокрутка со scroll-snap и блокировкой стрелок на краях.
- **Модальные окна** — подробная информация о тренере; закрытие по `Esc` / клику на оверлей, возврат фокуса.
- **FAQ-аккордеон** — single-open, плавное раскрытие, доступность через `aria-expanded` / `aria-controls`.
- **Анимированные счётчики** — статистика «накручивается» от нуля при попадании в зону видимости.
- **Scroll-reveal анимации** — плавное появление секций (IntersectionObserver).
- **Форма с валидацией и маской телефона** — автоформат `+7 (XXX) XXX-XX-XX`, инлайновые ошибки и сообщение об успехе.
- **Доступность** — семантические теги, корректная иерархия заголовков, `alt`, видимый focus, поддержка `prefers-reduced-motion`.
- **SEO** — meta-description, Open Graph, favicon, `theme-color`.

## 🧩 Разделы лендинга

Герой → О школе (анимированные показатели) → Для кого курс → Программа → Что вы получите → Как проходит обучение → Лаборатория R:ED LAB (фото) → Тренеры → Отзывы → Партнёры и лицензия → Тарифы + таблица сравнения → FAQ → Форма записи → Расширенный футер с контактами.

## 🛠️ Стек

`HTML5` · `CSS3` (Custom Properties, Grid, Flexbox, `clamp()`) · `JavaScript (ES6, IntersectionObserver)`

## 📁 Структура

```
.
├── index.html        # разметка
├── style.css         # стили + дизайн-токены (:root)
├── responsive.css    # медиа-запросы
├── script.js         # меню, scrollspy, прогресс/«наверх», карусель, модалки,
│                     # счётчики, scroll-reveal, FAQ, валидация + маска телефона
└── images/           # изображения, favicon.svg и слот lab.jpg
```

> **Фото лаборатории:** блок «Лаборатория R:ED LAB» ожидает файл `images/lab.jpg` (4:3). Пока файла нет, на его месте показывается аккуратный плейсхолдер — добавьте своё изображение, чтобы заменить его.

## 🚀 Запуск локально

Проект статический — достаточно открыть `index.html` в браузере. Для корректной работы относительных путей удобнее поднять локальный сервер:

```bash
# Python 3
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

или расширение **Live Server** в VS Code.

## 📦 Деплой на GitHub Pages

1. Запушьте репозиторий на GitHub.
2. **Settings → Pages → Build and deployment → Source: Deploy from a branch**.
3. Выберите ветку `main`, папку `/ (root)` и сохраните.
4. Через минуту сайт будет доступен по адресу `https://<username>.github.io/<repo>/`.
